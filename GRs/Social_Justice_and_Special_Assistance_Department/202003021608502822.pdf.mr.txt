| २.१ सदरील शासन निर्णयानुसार महाडीबीटी प्रणालीवर महाविद्यालय / अभ्यासक्रम मॅपिंग |
| (समावेश) करणे, महाविद्यालयनिहाय मंजूर अभ्यासक्रम व त्याची मंजूर प्रवेश क्षमता अपलोड करणे व ती |
| वेळीवेळी अद्ययावत करणे, त्याबाबत ऑनलाईन तपासणी करणे, भविष्यात प्रणालीवर नोंद झालेले |

| आल्यास, त्याअनुषंगाने महाडीबीटी प्रणालीवर आवश्यक ते बदल/ दुरुस्त्या करणे इ. जबाबदारी ही संबंधित |

| प्रवर्गातील विद्यार्थ्यांकरिता राबविण्यात येणाऱ्या शैक्षणिक शुल्क-परिक्षा शुल्क प्रतिपूर्ती योजनेची महाडिबीटी |

| बी.एड, बी.पीएड, एम.एड, एम.पीएड हे अभ्यासक्रम विभागांतर्गंत “महाईस्कॉल” प्रणालीवर “अव्यावसायिक |
| अभ्यासक्रम” म्हणून नोंदविण्यात आलेले होते. दरम्यान, उच्च व तंत्र शिक्षण विभागाच्या अनुक्रमे दिनांक |
| २४एप्रिल,२०१७ व दिनांक २१जानेवारी, २०१६ च्या अधिसूचनांन्वये सदरील अभ्यासक्रम हे “व्यावसायिक |
| अभ्यासक्रम” म्हणून घोषित करण्यात आलेले आहेत. त्यानुषंगाने सदर अभ्यासक्रमात शिकत असलेल्या |
| अनुसूचित जाती प्रवर्गातील विद्यार्थ्यांना “व्यावसायिक अभ्यासक्रम” म्हणून शिक्षण शुल्क- परिक्षा शुल्क |
| प्रतिपूर्ती योजनेचा लाभ देणे आवश्यक आहे. उच्च शिक्षण विभागाने सदरील अभ्यासक्रम “व्यावसायिक |
| अभ्यासक्रम” म्हणून महाडीबीटी प्रणालीवर मॅपिंग (समावेश) करुनही, तसेच सदरील अभ्यासक्रमांकरीता |
| शिक्षण शुल्क परिक्षा शुल्क प्रतिपूर्ती योजना अनुज्ञेय असूनही, महाडीबीटी प्रणालीवरील निर्माण झालेल्या |
| संभ्रमामुळे अनुसूचित जातीच्या विद्यार्थ्यांना महाडीबीटी प्रणालीवर सदर योजनेअंतर्गंत अर्ज नोंदणीकृत |
| करता येत नसल्याची तसेच संबंधित महाविद्यालयांना सदर अभ्यासक्रमांचे अर्ज नुतनीकरण करता येत |
| नसल्याची बाब निदर्शनास आली आहे. त्याचबरोबर, उच्च व तंत्रशिक्षण विभागामार्फत नव्याने |
| “अव्यावसायिक अभ्यासक्रम” हे “व्यावसायिक अभ्यासक्रम” म्हणून घोषित केलेल्या अभ्यासक्रमांबाबत |
| देखील याच प्रकारचे संभ्रम विद्यार्थी व महाविद्यालयांमध्ये निर्माण झालेला आहे. यामध्ये प्रामुख्याने राज्यभरात |
| राबविण्यात येणारे बीबीए, बीसीए, बीबीएम, बीएससी (कॅम्प्यु.सा.), एमएससी (कॅम्प्यु.सा.), एमसीए (कॉमर्स), |
| एमसीए (सायन्स) इ. अभ्यासक्रमांबाबत राज्य भरातील अनुसूचित जाती प्रवर्गातील विद्यार्थी व पालकांमध्ये |
| संभ्रम निर्माण झाल्याची बाब निदर्शनास आली आहे. या पार्श्वभूमीवर संदर्भीय दिनांक ३१.३.२०१६ रोजीच्या |
| शासन निर्णयाच्या अनुषंगाने सुधारीत सूचना निर्गमित करण्याची बाब शासनाच्या विचाराधीन होती. |

| शासन निर्णय : |
| ३. उच्च व तंत्र शिक्षण विभाग तसेच अन्य प्रशासकीय विभागामार्फत राज्यातील शासकीय व |
| अनुदानित महाविद्यालये, शासन मान्यता प्राप्त खाजगी विना अनुदानित व कायम विनाअनुदानित |
| महाविद्यालये / शिक्षण संस्थामधील “अव्यावसायिक अभ्यासक्रम" हे "व्यावसायिक अभ्यासक्रम” घोषित |
| केल्यास वा नव्याने “व्यावसायिक अभ्यासक्रमांचा समावेश केल्यास, अशा महाडिबीटी प्रणालीवर मॅपिंग |
| केलेल्या व्यावसायिक अभ्यासक्रमासाठी प्रवेशित अनुसूचित जाती प्रवर्गातील विद्यार्थ्यांना, शैक्षणिक शुल्क |
| परिक्षा शुल्क प्रतिपूर्ती योजनेचा लाभ अनुज्ञेय राहील. सदर शैक्षणिक लाभ हा, संबंधित प्रशासकीय विभागाने |
| महाडिबीटी प्रणालीवर व्यावसायिक अभ्यासक्रम ज्या शैक्षणिक वर्षापासून नोंद केलेला असेल, त्या शैक्षणिक |
| वर्षापासून लागू राहील. |
| ३.१ अनुसूचित जाती प्रवर्गातील विद्यार्थ्यांना राज्यातील शासकीय व अनुदानित महाविद्यालये, |
| शासन मान्यता प्राप्त खाजगी विना अनुदानित व कायम विनाअनुदानित महाविद्यालये / शिक्षण संस्थामधील |
| व्यावसायिक अभ्यासक्रमांकरीता शैक्षणिक शुल्क- परिक्षा शुल्क प्रतिपूर्ती योजनेचा लाभ घेण्यासाठी सक्षम |
| प्राधिकारी यांच्याकडील जात वैधता प्रमाणपत्र सादर करणे बंधनकारक राहील. |
| ३.२ अनुसूचित जाती प्रवर्गातील विद्यार्थ्यांना राज्यातील शासकीय व अनुदानित महाविद्यालये, |
| शासन मान्यता प्राप्त खाजगी विना अनुदानित व कायम विनाअनुदानित महाविद्यालये / शिक्षण संस्थामधील |
| व्यावसायिक अभ्यासक्रमांकरीता शैक्षणिक शुल्क- परिक्षा शुल्क प्रतिपूर्ती योजनेच्या लाभाकरीता, वरील |
| सुधारणांसह संदर्भीय दिनांक ३१.३.२०१६ च्या शासन निर्णयामधील उर्वरीत अटी व शर्ती लागु राहतील. |
| ३.३ सन २०१८-१९ या शैक्षणिक वर्षापासून अनुसूचित जाती प्रवर्गातील विद्यार्थ्यांना शैक्षणिक |
| शुल्क- परिक्षा शुल्क प्रतिपूर्ती योजनेच्या अंमलबजावणीबाबत महाडिबीटी प्रणालीवरील कार्यवाही ही संदर्भीय |
| दि.१२.१०.२०१८ व दि. ८.१.२०१९ च्या शासन निर्णय व शासन परिपत्रका नुसार करण्यात यावी. |
