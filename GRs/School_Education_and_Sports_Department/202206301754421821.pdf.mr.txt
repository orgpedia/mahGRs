| दि. १७ मे २०१७ चा शासन निर्णय तत्कालीन स्थितीतील अतिरिक्त शिक्षकांना सामावून |
| घेण्यासाठी काढण्यात आला. रात्रशाळेवर ३४ कोटी रूपयांचा अधिक वार्षिक खर्च दुबार शिक्षक कमी |
| केल्याने वाचू शकेल असे शासनास अपेक्षित होते. रात्रशाळेतील कार्यभार वाढवून अर्धवेळ अंशकालिक |
| शिक्षकांना पूर्णवेळ शिक्षक म्हणून अनुज्ञेय असलेल्या वेतनश्रेणीचे लाभ अनुज्ञेय राहील, असे या शासन |
| निर्णयात नमूद होते. राज्यात सध्या १७६ रात्रशाळा असून फक्त मुंबईत १५० हून अधिक रात्रशाळा व रात्र |
| कनिष्ठ महाविद्यालये आहेत. |
| उपरोक्त संदर्भ क्र.४ येथील शासन निर्णयान्वये रात्र शाळेतील शिक्षक शिक्षकेतर कर्मचा-यांच्या |
| सेवाशर्ती व अन्य बाबींसंदर्भात निर्णय घेण्याकरीता मा. मंत्री, शिक्षण यांच्या अध्यक्षतेखाली उच्च स्तरीय |
| समिती गठीत करण्यात आली होती. सदर समितीच्या बैठकीत दि.१७ मे,२०१७ रोजीच्या शासन निर्णयाच्या |
| अंमलबजावणीच्या अनुषंगाने रात्रशाळांच्याबाबत खालील बाबी निदर्शनास आल्या आहेतः |
| १) महाराष्ट्रातील खाजगी शाळातील कर्मचारी सेवा शर्ती नियम १९८१ मधील नियम २२(२) (ग) आणि |
| २३(१) (ब) नुसार दुबार काम करण्याची सवलत देण्यात आलेली आहे. या अधिनियमाशी उपरोक्त |
| शासन निर्णयातील तरतुदी विसंगत आहेत. |
| २) रात्रशाळांमध्ये एकूण १,०१० शिक्षक आणि ३४८ शिक्षकेतर कर्मचारी मिळून १,३५८ असे दुबार |
| कर्मचारी यांच्या सेवा समाप्त करण्यात आल्या. माध्यमिक रात्रशाळेतील ८६५ दुबार काम करणाऱ्या |
| शिक्षकांच्या सेवा समाप्त केल्यानंतर त्यांच्या जागी दिवस शाळेतील १७४ अतिरिक्त शिक्षकांचे |
| समायोजन करण्यात आले. शिक्षकच नसल्यामुळे बहुतांश रात्रशाळांमध्ये अध्यापनाचे काम सुरळीतपणे |
| होऊ शकले नाही. |

| ३) रात्र कनिष्ठ महाविद्यालयातील ११५ दुबार शिक्षकांना कमी करण्यात आले. त्यांच्या जागी |
| दिवसाच्या कनिष्ठ महाविद्यालतील अतिरिक्त शिक्षक उपलब्ध न झाल्यामुळे अध्यापनाचे काम पुर्णपणे |
| बंद झाले. ही बाब लक्षात आल्यानंतर रात्र कनिष्ठ महाविद्यालयातील दुबार शिक्षकांची सेवा सुरू ठेवून |
| त्यांचे वेतन पूर्वीप्रमाणे अदा करण्याचे आदेश शिक्षण उपसंचालक यांनी दिले. |
| ४) वरील आदेशानुसार पुन्हा दुबार शिक्षकांना कायम ठेवल्यामुळे रात्र कनिष्ठ महाविद्यालयाची प्रगती |
| समाधानकारक दिसत आहे. तथापि माध्यमिक रात्रशाळांमध्ये दुबार शिक्षक कमी करण्यात आल्यामुळे |
| या शाळांची पटसंख्या रोडावली आहे. तर गणित, विज्ञान आणि इंग्रजी या महत्त्वाच्या विषयांचे |
| अध्यापन बंद पडले आहे. |
| ५) दिवस शाळेतील अतिरिक्त शिक्षकांचे समायोजन रात्रशाळेत होऊ शकत नाही. अर्धवेळ अध्यापन |
| करणाऱ्या शिक्षकांना पूर्णकालिन शिक्षकांचे वेतन देणे कायद्याने अनुज्ञेय नाही. |
| ६) रात्रशाळेतील दुबार शिक्षकांना त्यांच्या अर्धवेळ सेवेसाठी अर्धवेळ वेतन श्रेणी देण्याचे शासन निर्णय |
| १५ जुलै १९७४ व २६ नोव्हेंबर १९८० नुसार निश्चित झाले आहे. |
| ७) राज्यशासनाने अधिनियम १९७७, सेवा नियम १९८१ (SS Code) निश्चित केले आहे. त्यातील |
| रात्रशाळांबाबत असलेल्या (Chapter II, Section VII, ५४.३) तरतुदी नुसार रात्रशाळांचे दैनंदिन |
| कामकाज अडीच तासांचे आहे. तर [Chapter II, Section VIl, ६७.१ (a) (IV)] नुसार शिक्षकांचा |
| कार्यभार अर्धवेळ राहील. सदर अधिनियमातील तरतुदींशी सुसंगत रात्रशाळांची व्यवस्था पूर्वीप्रमाणे |
| कायम करणे आवश्यक आहे. |
| ८) रात्रशाळांचा महाराष्ट्रातील इतिहास मोठा आहे. महात्मा फुले यांना मारण्यासाठी सनातन्यांनी |
| घातलेले मारेकरी त्यांना शरण गेले. त्यांना आणि अन्य कष्टकरी तरुणांना शिकवण्यासाठी पहिली |
| रात्रशाळा महात्मा फुले यांनी सुरू केली. या रात्रशाळेत महात्मा फुलेंच्या दिवस शाळेतील अनुभवी |
| शिक्षक सावित्रीबाई फुले यांच्या मदतीने रात्री पुन्हा म्हणजे दुबार पद्धतीने शिकवू लागले. त्यामुळे |
| अंशत: शाळा असूनही अनुभवी शिक्षकांचे गोळीबंद शिक्षण मिळू लागले. त्यातून धोंडिंबा नामदेव |
| कुंभार हा महापंडित झाला. |
| ९) डॉ. बाबासाहेब आंबेडकर यांनी परळ येथे राहत असताना सोशल सर्विस लीगच्या रात्रशाळेत |
| संस्थापक कामगार नेते ना. म. जोशी यांच्या आग्रहानुसार अर्धवेळ अध्यापन केले आहे. म्हणजे दुबार |
| पद्धतीने ते दोन वर्ष शिकवत होते. गिरणी कामगारांसाठी ही रात्रशाळा होती. त्यांनी स्वत: कुलाबा येथे |
| दिवसभर काम करण्याऱ्या मुलांसाठी मुकुंदराव आंबेडकर नाईट स्कूल सुरू केले. त्यात अनुभवी दुबार |
| शिक्षक नेमले. १ हजार रुपयांची वार्षिक ग्रँट दिली. मोफत पुस्तकांची व्यवस्था केली. |

| वरील सर्व बाबी विचारात घेऊन रात्रशाळांतील विद्यार्थ्यांना अनुभवी शिक्षकांकडून दर्जेदार शिक्षण |
| उपलब्ध करून देण्यासाठी व रात्रशाळांची गरज लक्षात घेऊन अधिनियम व वेगवेगळ्या समित्यांनी |
| केलेल्या शिफारशी यांनी निश्चित केलेली व्यवस्था कायम करण्यासाठी १७ मे २०१७ चा शासन निर्णय |
| अधिक्रमित करून याद्वारे खालील निर्णय घेण्यात येत आहे. |
| १) राज्यशासनाने महाराष्ट्र खाजगी शाळांतील कर्मचारी (सेवेच्या शर्ती) विनियमन अधिनियम, १९७७ व |
| महाराष्ट्र खाजगी शाळांतील कर्मचारी (सेवेच्या शर्ती) नियमावली, १९८१ व माध्यमिक शाळा संहिता |
| निश्चित केले आहेत. त्यातील रात्रशाळांबाबत असलेल्या (Chapter II, Section VII, ५४.३) तरतुदी |
| नुसार रात्रशाळांचे दैनंदिन कामकाज अडीच तासांचे राहील. तर [Chapter II, Section VII, ६७.१ |
| (a) (IV)] नुसार शिक्षकांचा कार्यभार अर्धवेळ राहील. |
| २) १७ मे, २०१७ च्या शासन निर्णयापूर्वी जे शिक्षक कार्यरत होते त्यापैकी नियमानुसार अनुज्ञेय |
| असलेल्या शिक्षकांना नेमणूका देण्यात येतील. अशा रात्रशाळा वर्गामध्ये किमान विहीत प्रवेशित |
| विद्यार्थी संख्या असणे आवश्यक राहील. तथापि, वित्तीय नुकसान टाळण्यासाठी रात्रशाळांमधील |
| रिक्त पदांवर दुबार शिक्षकांच्या नियुक्त्या करण्यापूर्वी जिल्ह्यातील/विभागातील अनुदानित अथवा |
| अंशत: अनुदानित दिवस अथवा रात्रशाळेत अतिरिक्त शिक्षक समायोजनासाठी उपलब्ध नाहीत |
| याची खात्री संबंधित विभागीय उपसंचालक यांनी करणे आवश्यक राहील. |
| ३) दिनांक २८ ऑगस्ट, २०१५ च्या शासन निर्णयानुसार विहित केल्याप्रमाणे विद्यार्थी संख्येवर |
| संचमान्यता तसेच शिक्षक संख्या निश्चित करण्यात यावी. |
| ४) मात्र वरीलप्रमाणे अनुज्ञेय होणारी शिक्षक संख्या अर्धवेळ शिक्षकांची राहील. |
| ५) रिक्त जागांवर नवीन दुबार शिक्षक नेमताना दिवस शाळेतील नियमित शिक्षकाचा किमान ३ वर्षांचा |
| अनुभव असणे आवश्यक आहे. |
| ६) मुद्दा क्र. २ मधील तरतूदींच्या अधिनतेने रात्रशाळांमधील (माध्यमिक व उच्च माध्यमिक) जे शिक्षक |
| अर्धवेळ काम करत होते व आहेत तथापि जे कोणत्याही दिवस शाळेत नाहीत अशा शिक्षकांच्या सेवा |
| यापुढेही कायम राहतील. ६ वर्षांहून अधिक काळ रात्रशाळेत सेवा केलेल्या अर्धवेळ शिक्षकांना दिवस |
| शाळेत (माध्यमिक व उच्च माध्यमिक) पवित्र पोर्टलमार्फत नियुक्तीद्वारे पूर्णकालीन सेवेची संधी |
| मिळाल्यास त्यांना नियुक्तीनंतर ३ वर्षांचा परिवीक्षाधीन कालावधी माफ होईल. |
| ७) विद्यार्थ्यांची गळती कमी करणे, शाळाबाह्य विद्यार्थ्यांना प्रवाहात आणणे व सभोवतालच्या |
| परिसरातील मुले शिक्षणापासून वंचित राहणार नाहीत याची जबाबदारी शिक्षकांची आहे. |
| ८) रात्रशाळेतील विद्यार्थ्यांना मोफत पाठ्यपुस्तके देण्यात येतील. |
