शालेय शिक्षण व क्रीडा विभाग
शासन निर्णय क्रमांकः निधिवि-२०२०/प्र.क्र.४९/प्रशिक्षण
मंत्रालय, मुंबई ४०० ०३२
दिनांक २१ जूलै, २०२०
वाचा:
१) शालेय शिक्षण व क्रीडा विभाग शासन निर्णय क्र.डायट ४५१६/(४०/१६)/प्रशिक्षण दिनांक १७/१०/२०१६ २) शालेय शिक्षण व क्रीडा विभाग शासन निर्णय क्र. डायट २०१८/प्र.क्र.७४/प्रशिक्षण दिनांक २३/८/२०१८ ३) शालेय शिक्षण व क्रीडा विभाग शासन निर्णय क्र. संकीर्ण २४१९/(५१/१९)/अर्थसंकल्प दिनांक ११/१२/२०१९ प्रस्तावना: शालेय शिक्षण व क्रीडा विभाग, शासन निर्णय दिनांक, १७/१०/२०१६ अन्वये विद्या प्राधिकरण व
संलग्न संस्थेची पुनर्रचना करण्यात आली असून व्यवसाय मार्गदर्शन व व निवड संस्था, मुंबई या संस्थेची पुनर्रचना करुन प्रादेशिक विद्या प्राधिकरण, मुंबई या कार्यालयाची स्थापना करण्यात आली आहे. शालेय शिक्षण व क्रीडा विभाग शासन निर्णय, दिनांक २३/८/२०१८ अन्वये शैक्षणिक तंत्रज्ञान कक्ष, वरळी या
कार्यालयाची प्रादेशिक विद्या प्राधिकरण, मुंबई या कार्यालयात विलीनिकरण करण्यात आलेले आहे.
पुनर्रचनेअंतर्गत निर्माण करण्यात आलेल्या प्रादेशिक विद्या प्राधिकरणामध्ये एक सूत्रता आणण्यासाठी प्रादेशिक विद्या प्राधिकरण, मुंबई व शैक्षणिक तंत्रज्ञान कक्ष, वरळी या कार्यालयातील कर्मचाऱ्यांच्या वेतन व इतर बाबीवरील खर्च भागविण्यासाठी तरतुद करण्यात येणारी पुढील लेखाशीर्षे मागणी क्र. ई-२, २२०२ सर्व साधारण शिक्षण, ०१ प्राथमिक शिक्षण, १०७, शिक्षक प्रशिक्षण ( ०० ) ( ०१ ) व्यवसाय मार्गदर्शन व माहिती केंद्र ( २२०२ ०६५६ ) मागणी क्र. ई-२, २२०२, सर्वसाधारण शिक्षण ( ०३ ) शैक्षणिक तंत्र विषयक कक्ष, ( ०३ ) ( ०१ ) शैक्षणिक तंत्र विषयक कक्ष ( अनिवार्य ) ( २२०२११५१ ) बंद
करण्याबाबतचा प्रस्ताव मा. आयुक्त (शिक्षण ) यांनी शासनास सादर केलेला आहे. पुर्नरचित प्रादेशिक विद्या प्राधिकरणास वेतन व इतर उद्दिष्टांवरील खर्च भागविण्यासाठी सन २०२०-२०२१ करीता वितरीत
करण्यात येणाऱ्या अनुदानाकरीता सांकेतांक २२०२१११५ याखाली एकत्रित तरतुद करण्यात आली असून या तरतुदीमधून वर नमूद केलेल्या कार्यालयावरील खर्च भागविण्यास मान्यता देण्याची बाब शासनाच्या विचाराधीन होती.

शासन निर्णय : पुर्नरचित प्रादेशिक विद्या प्राधिकरण, मुंबई व शैक्षणिक तंत्रज्ञान कक्ष, वरळी यांच्या वेतन व इतर उद्दिष्टांवरील खर्च मागणी क्र. ई- २, २२०२, सर्वसाधारण शिक्षण, ००३, प्रशिक्षण, ( ०१ ) शिक्षकांचे प्रशिक्षण, ( ०१ ) ( ०१ ) शिक्षकांचे प्रशिक्षण (अनिवार्य ) ( २२०२ १११५ ) या पुढे या लेखाशीर्षाखालील तरतुदीमधून भागविण्यास शासन मान्यता देण्यात येत आहे.

अधिकारी बालचित्रवाणी, पुणे हे असतील असे संदर्भ क्र. ३ अन्वये घोषित करण्यात आले आहे.
