प्रस्तावना - राज्यातील वाहनांची संख्या दिवसेंदिवस वाढत असुन त्यामुळे वाहतुक कोंडी व इतर समस्या निर्माण होत आहेत. वाहतुक नियमांचे उल्लंघन करणाऱ्यांकडुन दंड वसुलीदरम्यान वाहतुक पोलीस कर्मचारी/अधिकारी यांना अडचणींना सामोरे जावे लागत आहे. त्यामुळे वाहतुक समस्यांवर अंकुश ठेवणे व कसुरदार व्यक्तींवर प्रभावी तथा पारदर्शकपणे कारवाई करण्यासाठी राज्यामध्ये ई-चलान प्रकल्प राबविण्यास व त्यासाठी येणाऱ्या रु.५९,१२,३१,५०७/- एवढ्या निधीच्या खर्चास प्रशासकीय मान्यता शासन
निर्णय, गृह विभाग दि.०३.०२.२०१८ नुसार देण्यात आली आहे. ई-चलान प्रकल्पाच्या निविदा प्रक्रीयेतुन पात्र ठरलेल्या पुरवठादाराच्या निवडीस उच्चस्तरीय शक्ती प्रदत्त समितीने दि.०७.०९.२०१८ च्या बैठकीत मान्यता दिलेली आहे. त्यानुसार महाराष्ट्र शासनाच्या वतीने अपर पोलीस महासंचालक (वाहतुक) आणि मे.
टेलिकम्युनिकेशन्स कन्सल्टंट्स इंडिया लिमिटेड यांचे दरम्यान दि.११.१२.२०१८ रोजी पाच वषांचा करार करण्यात आलेला आहे. सदर करार हा कार्यारंभ आदेशापासून पाच वषांसाठी आहे. तसेच यशस्वी अंमलबजावणी व चांगल्या सेवा विचारात घेऊन वाहतूक पोलीस हा करार पुढील २ वषे वाढवू शकतात अशी तरतूद करारात आहे. संदर्भ क्र.८ येथील पत्राद्वारे मे. टेलिकम्युनिकेशन्स कन्सल्टंट्स इंडिया लिमिटेड यांना कार्यादेश देण्यात आले आहेत.

(1) Data Migration Cost:- (A) Data of the existing One State One eChallan project 1s required to be migrated to the NIC’s eChallan system, district by district.
(B) It is decided that responsibility of migration of legacy data to NIC shall be in the scope of existing Service Provider TCIL, including any cost associated with the same.
(2) eCourt’s integration :- (A) One State One eChallan system is not integrated with eCourt system, however NIC’s eChallan system should be integrated with eCourt.
(B) All the responsibility of the same shall be of the NIC.
(3) NIC’s eChallan implementation:- (A) It is decided that NIC’s eChallan system should be implemented in phase wise manner. (B) Completion of phase wise implementation for NIC’s eChallan system should be only considered after successful migration of respective legacy data, integration with eCourt system and whenever NIC is ready with all the approved functionalities of ADG (Traffic) office in existing O.S.O.C. System. (C) Modalities for phase wise implementation should be decided by the PIC.
(4) Extension to existing Service Provider (TCIL) for One State One eChallan project:- (A) The contract agreement signed with the existing service provider (TCIL) for ‘One State One eChallan’ project has provision for extension for maximum 2 years post the original contract end date. (B) It 1s decided that existing service provider will be given extension for maximum two (2) years (from O7 Mar 2024 to 06 Mar 2026), OR till the time NIC’s eChallan system is implemented statewide in a phased manner, whichever is earlier.
