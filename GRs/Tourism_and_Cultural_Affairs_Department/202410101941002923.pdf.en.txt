10) The Collector, as the controlling officer, should ensure that no money has been taken from the Urban Development Department, Planning Department, District Planning Board and any other scheme for various works in this temple in the past.
11) If funds have been sanctioned for other works in the same project from any department other than the Tourism Department, the works in this project should not be taken up with the Government.
12) The Collector and the Project Implementation Mechanism should ensure that the work is not duplicated and certified.
13) The project execution mechanism should be done by the Public Works Department.
14) Work on the project should be done by assigning a priority number. Do not start all the work together. So that tourists, devotees will not be inconvenienced.
15) In the course of project work, the government should take action as per the prevailing rules with regard to tender matters. The project implementation agency should take care of this.
