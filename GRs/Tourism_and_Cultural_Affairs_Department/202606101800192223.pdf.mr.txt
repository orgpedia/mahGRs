| वाचा :- सांस्कृतिक कार्य संचालनालय, मुंबई यांचे पत्र क्र.सां.का.सं./१७३७, |
| दिनांक २७ मे,२०२६. |
|  |
| प्रस्तावना :- |
| भारताच्या सांस्कृतिक व आध्यात्मिक वारशामध्ये सोमनाथ मंदिराला अत्यंत महत्तवाचे स्थान असल्यामुळे |
| "सोमनाथ स्वाभिमान पर्व १००० वषे अखंड आस्था" हा कार्यक्रम दिनांक ११ जानेवारी २०२६ ते ११ जानेवारी २०२७ |
| या कालावधीत देशभर साजरा करण्यात येत असल्याचे संस्कृती मंत्रालय, भारत सरकार यांनी त्यांच्या दि. ०३ |
| फेब्रुवारी, २०२६ रोजीच्या पत्रान्वये कळविले आहे. या कालावधीत सर्व राज्य आणि केंद्रशासित प्रदेशांच्या |
| सहभागाने समन्वित पध्दतीने वर्षभर चालणारा राष्ट्रीय उत्सव म्हणून आयोजित करण्यात यावा असे निदेश दिलेले |
| आहेत. या अनुषंगाने महाराष्ट्र राज्यात विविध धार्मिक, सांस्कृतिक व जनजागृतीपर उपक्रम राबविणे प्रस्तावित |
| आहे. सोमनाथ मंदिराच्या इतिहासाबाबत जनजागृती करणे, भारतीय सांस्कृतिक वारसा व श्रध्दा वृच्ध्दगत करणे, |
| युवकांमध्ये आध्यात्मिक व सांस्कृतिक मूल्ये रुजविणे, राज्यातील तीर्थस्थळांचा विकास व पर्यटनाला चालना देणे हे |
| या कार्यक्रमाचे उद्दिष्ट्टे राहील. |
| सदर संकल्पनेवर आधारित पहिला सांस्कृतिक कार्यक्रम दिनांक ११ मे, २०२६ रोजी श्री धाकलेश्वर मंदिर |
| (पटांगण), महालक्ष्मी कपांऊंड, मुंबई येथे आयोजन करण्यासाठीच्या खर्चास प्रशासकीय व वित्तीय मान्यता देण्याची |
| विनंती संचालनालयाने त्यांच्या संदर्भाधीन पत्रान्वये केली होती. त्यानुषंगाने सदर सांस्कृतिक कार्यक्रमाच्या |
| आयोजन खर्चास प्रशासकीय व वित्तीय मान्यता देण्याची बाब शासनाच्या विचाराधीन होती. |
|  |
| शासन निर्णय :- |
| "सोमनाथ स्वाभिमान पर्व १००० वषे अखंड आस्था" निमित्त श्री धाकलेश्वर मंदिर (पटांगण), महालक्ष्मी |
| कपांऊंड, मुंबई येथे दिनांक ११ मे, २०२६ रोजी आयोजित करण्यात आलेल्या सांस्कृतिक कार्यक्रमास व त्यावरील |
| खर्च रुपये २५,००,०००/- (अक्षरी रुपये पंचवीस लक्ष फक्त) इतक्या निधीस या शासन निर्णयान्वये प्रशासकीय व |
| वित्तीय मान्यता देण्यात येत आहे. |
