मंत्रालय, मुंबई - ४०० ०३२.
दिनांक :- २६ ऑक्टोबर, २०२०
संदर्भ : १) शासन निर्णय क्रमांक :- संकीर्ण-२०१६/प्र.क्र.२५९/नवि-३३, दिनांक ३० नोव्हेंबर, २०१७
प्रस्तावना : मा. राष्ट्रीय हरित लवादामध्ये दाखल मुळ अर्ज क्र. ५९३/२०१७ (पर्यावरण सुरक्षा समिती विरुध्द केंद्र शासन व सर्व राज्य शासन) या प्रकरणी मा. लवादाने दिनांक २८.०८.२०१९ रोजी दिलेल्या निदेशानुसार नागरी भागामध्ये निर्माण होणाऱ्या सांडपाण्यावर १०० टक्के प्रक्रिया करणे सर्व नागरी स्थानिक संस्थांना बंधनकारक करण्यात आले आहे. या अनुषंगाने राज्यातील एकूण ३९१ नागरी स्थानिक स्वराज्य संस्थांमध्ये एकूण अंदाजे ९७५८ एमएलडी इतक्या सांडपाण्यावर प्रक्रिया करण्याची गरज असून यापैकी अंदाजे ७७४७ एमएलडी सांडपाण्यावर प्रक्रिया करण्याची क्षमता शासनाच्या विविध योजनांमधून/नागरी स्थानिक संस्था निधीतून निर्माण करण्यात आली आहे/ निर्माणाधीन आहे. उर्वरीत २०११ एमएलडी मलप्रक्रिया क्षमता संबंधीत नागरी स्थानिक संस्थेकडून उभारुन घेण्याची बाब शासनाच्या विचाराधीन होती.
शासन निर्णय :
मा. राष्ट्रीय हरित लवादामध्ये दाखल मुळ अर्ज क्र.५९३/२०१७ (पर्यावरण सुरक्षा समिती विरुध्द केंद्र शासन व सर्व राज्य शासन) या प्रकरणी मा. लवादाने दिनांक २८.०८.२०१९ रोजी दिलेल्या निदेशानुसार नागरी भागामध्ये निर्माण होणाऱ्या सांडपाण्यावर १०० टक्के प्रक्रिया करणे सर्व नागरी स्थानिक संस्थांना बंधनकारक करण्यात आले आहे. त्याअनुषंगाने, राज्याच्या नागरी भागात निर्माण होणा-या सांडपाण्यावर प्रक्रिया करण्याबाबतचा कृती आराखडा व धोरण खालीलप्रमाणे विहित करण्यात येते आहे.

| १.३.३ त्यानुसार नगरपंचायत क्षेत्रांमध्ये उपलब्ध पाणीपुरवठा, निर्माण होणारे सांडपाणी बघता |
| आवश्यक असणारी यंत्रणा ही तात्काळ उभी करणे आवश्यक नाही. सबब, विकेंद्रीत सांडपाणी |
| व्यवस्थापन (Decentralized Septage Management) व्यवस्था डिसेंबर, २०२० अखेर निर्माण |
| करण्यात यावी. यासाठी आवश्यक असणारा खर्च हा स्वच्छ भारत अभियानातून करण्यात यावा. |
|  |
| १.४ सांडपाणी पुनर्चक्रीकरण व पुनर्वापर आराखडा : |
|  |
| १.४.१ राज्यामध्ये नव्याने मलनिस्सारण प्रकल्पाचा डीपीआर तयार करताना, त्यामध्ये प्रक्रियायुक्त |
| सांडपाण्याचे पुनर्चक्रीकरण व पुनर्वापर आराखडयाचा समावेश डीपीआर मध्ये करावा. |
| तद्नंतरच, नवीन प्रकल्पास मंजूरी प्रदान करण्यात यावी. |
| १.४.२ सध्या प्रगतीपथावर असलेल्या मलनिस्सारण प्रकल्पांची कामे पूर्ण होण्यापूर्वी, प्रक्रियायुक्त |
| सांडपाण्याचे पुनर्चक्रीकरण व पुनर्वापर आराखडा तयार करून त्यास शासनाची मान्यता घेण्यात |
| यावी. तसेच प्रकल्प पूर्ण होताच त्याची अंमलबजावणी करण्यात यावी. |
| १.४.३ राज्याच्या नागरी भागातील ज्या मलनिस्सारण प्रकल्पांची कामे पूर्ण झाली आहेत. अशा नागरी |
| स्थानिक स्वराज्य संस्थांनी पुढील ६ महिन्यांमध्ये प्रक्रियायुक्त सांडपाण्याचे पुनर्चक्रीकरण व |
| पुनर्वापर आराखडयास शासनाची मान्यता घेऊन त्याची अंमलबजावणी करावी. |
|  |
| १.५ नागरी भागातील निर्माण होणाऱ्या सांडपाण्यावरील प्रक्रियेबाबत आराखड्याच्या अंमलबजावणीसाठी |
| कालमर्यादा: |
|  |
| १.५.१ सदर आराखडयांतर्गत राज्यातील बृहन्मुंबई महानगरपालिका इतर १७ अमृत शहरे, १८७ |
| नगरपरिषदा व नगरपंचायत क्षेत्रांमध्ये सांडपाणी प्रक्रिया व्यवस्था निर्माण करणे आवश्यक आहे. |
| सदर कामाचे स्वरुप व्यापक आहे. हे काम वेळ लागणारे आहे. |
| १.५.२ त्यानुसार सदर प्रकल्पांच्या अनुषंगाने प्रत्यक्षात सर्वेक्षण करणे, डीपीआर तयार करणे, त्यास |
| सक्षम प्राधिकरणाची तांत्रिक मान्यता घेणे, प्रत्यक्षात प्रकल्प मंजूर करणे यासाठी किमान १ |
| वर्षाचा कालावधी लागणे अपेक्षित आहे. |
| १.५.३ तसेच प्रकल्पांस मंजूरी दिल्यानंतर त्याचे निविदा प्रपत्र तयार करणे, प्रत्यक्ष निविदा प्रक्रिया |
| राबविणे, एवढ्या मोठ्या प्रमाणात गुणवत्तापूर्ण मलप्रक्रिया व्यवस्था उभारण्यासाठी मोठया |
| प्रमाणात तांत्रिक मनुष्यबळ, तांत्रिकदृष्ट्या सक्षम कंत्राटदार उपलब्ध होणे या व अन्य बाबींमुळे |
| प्रत्यक्षात प्रकल्पांची कामे सुरु करण्यासाठी आवश्यक कालावधी लागणार आहे. |
| १.५.४ प्रत्यक्षात प्रकल्पाच्या कामास सुरवात झाल्यानंतर किमान २४ महिन्यांचा कालावधी प्रकल्प |
| कार्यान्वित होण्यासाठी लागणे अपेक्षित आहे. या सर्व बाबींचा व्यवहार्य विचार करूनच सदर |
| आराखडा पूर्ण करण्यासाठी शासनाने डिसेंबर, २०२४ ही कालमर्यादा प्रस्तावित केली आहे. |
|  |
| ०२. हा आराखडा राष्ट्रीय हरित लवादास सादर करण्यात यावा. त्याच्या अमंलबजावणीसाठी डिसेंबर, |
| २०२४ पर्यंत मुदतवाढ देण्याची राष्ट्रीय हरित लवादास विनंती करण्यात यावी. |

०४. नगरपंचायत क्षेत्रामध्ये विकेंद्रीत सांडपाणी व्यवस्थापन (Decentralized Septage Management) व्यवस्था निर्माण करण्यासाठी आवश्यक निधी हा केंद्र शासन पुरस्कृत स्वच्छ भारत अभियानातून उपलब्ध करून देण्यात यावा. यासाठी आवश्यक खर्च हा नगरविकास विभागाच्या उपलब्ध नियतव्ययातून करण्यात यावा. आवश्यकता भासल्यास या प्रयोजनार्थ अरितिक्त नियतव्यय वित्त विभागाने उपलब्ध करून द्यावा.
०५. बृहन्मुंबई महानगरपालिका क्षेत्रामध्ये मलप्रक्रिया व्यवस्था निर्माण करण्यासाठी आवश्यक निधी हा बृहन्मुंबई महानगरपालिकेने उभारावा.
०६. हा शासन निर्णय नियोजन विभाग व वित्त विभागाकडून प्राप्त अभिप्राय विचारात घेऊन त्याबाबतच्या मंत्रीमंडळ निर्णयास अनुसरुन निर्गमित करण्यात येत आहे.
०७. सदर शासन निर्णय महाराष्ट्र शासनाच्या www.maharshtra.gov.in या संकेतस्थळावर उपलब्ध करण्यात आला असून त्याचा संकेतांक २०२०१०२६१५२१२०८४२५ असा आहे. हा आदेश डिजीटल स्वाक्षरीने साक्षांकित करुन काढण्यात येत आहे.
