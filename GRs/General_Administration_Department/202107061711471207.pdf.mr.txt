| भूषण, पद्मविभूषण) शिफारशी केंद्र शासनाकडे पाठवावयाच्या आहेत. पद्म पुरस्कार हे सर्वोच्च नागरी |
| पुरस्कारांपैकी एक असून ते ते अतिशय प्रतिष्ठेचे असतात. हे पुरस्कार विविध क्षेत्रांमधील उल्लेखनीय सेवेबद्दल |
| दिले जातात. या बाबतचे अधिनियम आणि नियम केंद्र शासनाच्या गृहमंत्रालयाच्या |
| www.padmaawards.gov.in या संकेतस्थळावर देण्यात आले आहेत. त्यांचे कृपया अवलोकन करावे. |
| २. पद्म पुरस्कारांकरिता शिफारशी करताना केंद्र शासनाने पुढीलप्रमाणे निकष सुचविले आहेत: |
| १) शिफारस करताना संबंधित क्षेत्रातील अत्युत्कृष्ट कार्य (Excellence Plus) हा निकष असावा. या |
| उल्लेखनीय कार्यामध्ये समाजसेवा हाही एक घटक असावा. |
| २) पद्म पुरस्कार हा भारतातील दुस-या क्रमांकाचा सर्वोच्च नागरी सन्मान असल्याने मान्यवरांची |
| शिफारस करताना शिफारस केलेल्या मान्यवरास त्याच्या कार्यक्षेत्रात यापूर्वी कोणते राष्ट्रीय/राज्य पुरस्कार |
| मिळाले आहेत काय, याचा विचार व्हावा. |
| ३) पुरस्कारासाठी नावे सुचवताना समाजातील अनुसूचीत जाती, अनुसूचीत जमाती, इतर मागास वर्ग, |
| महिला, अपंग व कमकुवत घटकांमधील प्रतिभासंपन्न व्यक्तींचाही विचार व्हावा. |
| ४) मरणोत्तर पुरस्काराची शक्यतो शिफारस करू नये. मात्र उच्चतम गुणवत्तेच्या अपवादात्मक |
| प्रकरणांमध्ये जानेवारी, २०२० नंतर निधन झालेल्या व्यक्तींची शिफारस करता येईल. त्यापूर्वी निधन झालेल्या |
| व्यक्तींची पद्म पुरस्कारासाठी शिफारस करू नये. |
| ५) पद्म भूषण पुरस्कारासाठी मान्यवरांची शिफारस पद्म श्री पुरस्कार प्राप्त केल्यानंतर ५ वर्षांनी करता |
| येईल. तसेच पद्म विभूषण पुरस्कारांसाठी मान्यवरांची शिफारस पद्म भूषण प्राप्त केल्यानंतर ५ वर्षांनी करता |
| येईल. तथापि ही मर्यादा अपवादात्मक प्रकरणांमध्ये शिथील करण्याचा विचार करता येईल. |
| ६) शासकीय सेवेतील तसेच सार्वजनिक उपक्रमातील डॉक्टर व वैज्ञानिक वगळता अन्य कर्मचारी |
| अधिकारी पद्म पुरस्कारासाठी पात्र नाहीत. |
| उपरोक्त निकषांच्या आधारे शिफारसयोग्य असतील असे प्रस्ताव जिल्हाधिकारी, विभागीय आयुक्त व |
| मंत्रालयीन प्रशासकीय विभागांनी या विभागाकडे दिनांक ६ ऑगस्ट, २०२१ पर्यंत निश्चितपणे पाठवावेत. |
| जिल्हाधिकारी व विभागीय आयुक्त वगळता, सर्व क्षेत्रीय कार्यालयांनी त्यांचे प्रस्ताव संबंधित मंत्रालयीन |
| विभागामार्फत पाठवावेत. |
| पुरस्कारासाठी प्रस्ताव पाठविताना खालील सूचना विचारात घ्याव्यात. |
| अ) पात्र व्यक्तींचा शोध घेताना वृत्तपत्रे किंवा इतर कोणत्याही प्रसिद्धी माध्यमांद्वारे जाहिराती देऊन नावे |
