डॉ. बाबासाहेब आंबेडकर आंतरराष्ट्रीय विमानतळ
नागपूर (॥-॥५) सह WA शासनाकडून विकसित होणा-या विमानतळांच्या Request for Proposal (RFP), Concession Document (CD) इत्यादी दस्ताऐवजांना मंजुरी देण्यासाठी Project Monitoring & Implementation Committee (PMIC) समिती नव्याने गठित करण्याबाबत.
महाराष्ट्र शासन
सामान्य प्रशासन विभाग
शासन निर्णय क्र. मविवि-१२१७/प्र.क्र.३००/विमान-१ (२८-ओ)
हुतात्मा राजगुरु चौक, मादाम कामा रोड,
मंत्रालय, मुंबई- 32.
दिनांक:- २४ नोव्हेंबर, २०२५
वाचा- १.झासन निर्णय क्र. मविवि-१२१७/प्र.क्र.3००/२८-अ, दिनांक ११ डिसेंबर, २०१७.
२. मा.मुख्य सचिव यांच्या अध्यक्षतेखालील दि.०७.११.२०२५ रोजी झालेल्या Project Monitoring & Implementation Committee (PMIC) च्या बैठकीचे दि.१०.११.२०२५ रोजीचे इतिवत्त.
प्रस्तावना:- डॉ.बाबासाहेब आंबेडकर आंतरराष्ट्रीय विमानतळ, नागपूर (MIHAN) सह राज्य
शासनाकडून विकसित होणा-या विमानतळांच्या Request for Proposal (RFP), Concession
Document (CD) इत्यादी दस्ताऐवजांना मंजुरी देण्यासाठी Project Monitoring & Implementation
Committee (PMIC) ची स्थापना शासन निर्णय, सामान्य प्रशासन विभाग,दिनांक ११ डिसेंबर २०१७
अन्वये करण्यात आली आहे.
मा.मुख्य सचिव यांच्या अध्यक्षतेखालील दि.०७.११.२०२५ रोजी झालेल्या Project
Monitoring & Implementation Committee (PMIC) च्या बैठकीमध्ये, सदर समितीमध्ये “अध्यक्ष व
व्यवस्थापकीय संचालक, मिहान इंडिया लि., नागपूर” यांचा पदसिध्द सदस्य (Ex-officio member)
म्हणून समावेश करण्याचे निदेश देण्यात आले आहेत. सदर निदेशानुसार संदर्भाधीन क्र.१ येथील
शासन निर्णय अधिक्रमित करण्याची बाब शासनाच्या विचाराधीन होती.
शासन निर्णय-
डॉ.बाबासाहेब आंबेडकर आंतरराष्ट्रीय विमानतळ, नागपूर (MIHAN) सह
राज्यझासनाकडून विकसित होणा-या विमानतळांच्या Request for Proposal (RFP), Concession
Document (CD) इत्यादी दस्ताऐवजांना मंजुरी देण्यासाठीच्या Project Monitoring &
Implementation Committee (PMIC) मध्ये “अध्यक्ष व व्यवस्थापकीय Wao, मिहान इंडिया
लि., नागपूर” यांचा पदसिध्द सदस्य (Ex-officio member) म्हणून समावेश करुन सदर समिती
नव्याने गठित करण्यात येत आहे.
