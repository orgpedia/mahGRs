वाचा : जलसंपदा विभाग, शासन निर्णय क्र. जविप्र २०१५/(४५/१५)/सुधा-१/जवि, दि.०४ एप्रिल, २०१६.
प्रस्तावना:
संदर्भाधीन शासन निर्णय दि.०४.०४.२०१६ अन्वये अधीक्षक अभियंता, कोयना (विवयां) उभारणी मंडळ, सातारा CO Code (l००८२) हे कार्यालय दि.३०.०४.२०१६ पासून बंद करण्यात आले आहे. तसेच सदर मंडळांतर्गत कार्यरत असलेली उभारणी विभाग (विवयां) क्र.१ अलोरे व उभारणी विभाग (विवयां ) क्र. ३ कोयनानगर ही विभागीय कार्यालये दि.०१.०५.२०१६ पासून अधीक्षक अभियंता, उभारणी मंडळ, कोल्हापूर या मंडळाच्या नियंत्रणाखाली वर्ग करण्यात आली आहेत. परंतु शासनाकडून प्राप्त अनुदान अधीक्षक अभियंता, कोयना (विवयां ) उभारणी मंडळ, सातारा या बंद झालेल्या कार्यालयाच्या नियंत्रणाखालील (CO Code l००८२) CRC Code ४८०१-३४५५-५३ मोठी बांधकामे हया संगणक सांकेतांक क्रमांकावर जमा होत आहे.
सदर मंडळ कार्यालय बंद असल्याने प्राप्त अनुदानाचा वापर करता येत नाही. त्यामुळे सदरचे अनुदान शासनास समर्पित करावे लागते. अधीक्षक अभियंता, कोयना (विवयां) उभारणी मंडळ, सातारा (सध्या बंद झालेले मंडळ) यांचे नियंत्रणाखालील CO Code l००८२ वरील कोयना धरण पायथा विद्युतगृह (डावा तीर) - ४८०१३४५५-५३ मोठी बांधकामे हा संगणक सांकेतांक अधीक्षक अभियंता, उभारणी मंडळ, कोल्हापूर यांचे CO Code 1००८० वर वर्ग करण्यात आल्यास सदर सांकेतांकावरील खर्च हा कोयना धरण पायथा विद्युतगृह (डावा तीर) या प्रकल्पाच्या कामांसाठी करणे सुलभ होईल. त्यानुषंगाने (CO Code l००८२) CRC Code ४८०१-३४५५-५३ मोठी बांधकामे हया संगणक सांकेतांक करीता अधीक्षक अभियंता, उभारणी मंडळ, कोल्हापूर यांना "नियंत्रक अधिकारी" घोषित करण्याची बाब शासनाच्या विचाराधीन होती. त्यानुषंगाने खालीलप्रमाणे निर्णय घेण्यात येत आहे.

शासन निर्णय क्रमांकः बजेट २०१९/प्र.क्र.१००/जवि
शासन निर्णय :
अधीक्षक अभियंता, कोयना (विवयां) उभारणी मंडळ, सातारा या बंद झालेल्या कार्यालयाच्या नियंत्रणाखालील (CO Code l००८२) वरील कोयना धरण पायथा विद्युतगृह (डावा तीर ) CRC Code ४८०१-३४५५-५३ मोठी बांधकामे करीता सर्व अनुषंगिक कार्यवाहीकरिता अधीक्षक अभियंता, उभारणी मंडळ, कोल्हापूर यांना "नियंत्रक अधिकारी" घोषित करण्यात येत आहे.
