यांनी ३६, सहायक अनुदाने (वेतन) या उद्दीष्टाकरिता बिम्स प्रणालीवर निधी उपलब्ध करुन दिलेला

रु.१२२५७.२६४ (रुपये एक कोटी बावीस लक्ष सत्तावन्न हजार दोनशे चौसष्ठ फक्त) इतका निधी
पहिला हप्ता म्हणून वितरित करण्यास शासन मान्यता देत आहे.

| ११९८/प्र.क्र.५८१/६१ दि.२३ सप्टेंबर, १९९८ व शासन परिपत्रक, ग्राम विकास व जलसंधारण विभाग |
| क्रमांक प्रशिक्षण २०००/प्र.क्र.८५९/४४, दि.१२ जून २००१ अन्वये निर्गमित करण्यात आलेल्या |
| सूचनांच्या अनुंषगाने सर्व संबंधित उपायुक्‍त (आस्था) यांनी सदर पंचायत राज प्रशिक्षण केंद्रास दरमहा |
| अनुदान अदा करण्याबाबत त्वरीत कार्यवाही करावी. तसेच संदर्भ क्र.१ मध्ये दर्शविलेल्या शासन |
| निर्णयातील परिच्छेद क्रमांक १२ मधील अटीनुसार केंद्रानी खर्च करावा. |

| ३. सदर निधी वितरणाकरिता तसेच निधी ज्या बाबींकरीता मंजूर करण्यात आला आहे, त्याच |
| बाबींवर खर्च करण्यात येईल याची काटेकोर अंमलबजावणी करण्याची जबाबदारी संबंधित उपायुक्‍त |
| (आस्था), विभागीय आयुक्‍त कार्यालय यांची असेल. तसेच सदर निधीचे उपयोगिता प्रमाणपत्रही |

| सरपंच, सचिव, अशासकीय सदस्य इत्यादींना प्रशिक्षण- राज्यक्षेत्र, ३१, सहायक अनुदान (वेतनेतर) व |

| बाबीवर प्रत्यक्ष किती अनुदानाची आवश्‍यकता आहे याची खातरजमा करुन त्यांना आवश्‍यक तेवढी |

| जाणार नाही याची दक्षता घ्यावी. तसेच प्रशिक्षण केंद्राने वेतन/वेतनेतरकरिता दरमहा किती अनुदान |
| आवरऱ्यक आहे याची पूर्वसूचना संबंधित आयुक्‍त कार्यालयास द्यावी. त्याप्रमाणे आयुक्‍त कार्यालयाने |
| वेतन/वेतनेतर बाबींकरिता अनुदान वितरीत करावे. आवश्‍यक असलेल्या अनुदानाबाबत पूर्वसूचना न |
| दिल्यास त्याची संपूर्ण जबाबदारी संबंधित केंद्राच्या प्राचार्यांची असेल. |
