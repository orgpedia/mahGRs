| केंद्रीय कोळसा मंत्रालय, भारत सरकार यांनी त्यांच्या दिनांक १८/१०/२००७ रोजीच्या |
| आदेशान्वये नवीन कोळसा वितरण धोरण तसेच त्यासंदर्भातच्या मार्गदर्शक सूचना जाहीर केल्या आहेत. |
| त्यास अनुसरुन राज्यातील लघू व मध्यम उद्योग उपक्रमांना शासन नियुक्त यंत्रणेमार्फत वितरीत |
| करण्यात येणा-या कोळशाच्या वाटपाबाबत दिनांक १२/०७/२०१६ रोजीच्या शासन निर्णयान्वये सुधारीत |
| मार्गदर्शक तत्त्वे निर्गमित करण्यात आली आहेत. त्यानुसार राज्यातील वार्षिक ४२०० मे. टन वापर |
| असणा-या लघू व मध्यम उपक्रमांना कोळसा वितरण करण्यासाठी सविस्तर निर्देश देण्यात आले आहेत. |
| तद्नंतर केंद्रीय कोळसा मंत्रालयाने त्यांच्या दिनांक २७/०९/२०१६ रोजीच्या आदेशान्वये दिनांक |
| १८/१०/२००७ रोजीच्या धोरणामध्ये सुधारणा केली असून त्याअन्वये राज्यातील वार्षिक १०,००० मे. टन |
| वापर असणा-या लघू, मध्यम व इतर उपक्रमांना राज्य नियुक्त यंत्रणेमार्फत कोळसा वितरण करण्यासाठी |
| मंजूरी देण्यात आली आहे. त्यास अनुसरुन तसेच विभागाकडे प्राप्त झालेली निवेदने तसेच शासन नियुक्त |
| यंत्रणेचे अभिप्राय विचारात घेता, यासंदर्भात अधिक पारदर्शक पद्धतीने कोळसा वाटप करणेकरीता यापूर्वी |
| दिनांक १२/०७/२०१६ रोजीच्या शासन निर्णयान्वये निर्गमित मार्गदर्शक तत्त्वांमध्ये आवश्यक सुधारणा |
| करण्याची बाब शासनाच्या विचाराधीन होती. |
| शासन निर्णयः |
| १. संदर्भ क्र.०३ येथील शासन निर्णय क्र. कोळसा-२०१४/प्र.क्र.२७५/उद्योग-७, दिनांक |
| १२/०७/२०१६ मध्ये “राज्यातील वार्षिक ४,२०० मे. टन वापर असणा-या लघू व मध्यम उपक्रमांना |
| कोळसा वितरण करण्यासाठी शासन स्वतंत्र आदेशाद्वारे शासन नियुक्त यंत्रणेची विहीत कालावधीसाठी |
| नियुक्ती करेल” अशी तरतूद आहे. |

शासन निर्णय क्रमांकः कोळसा-२०२०/प्र.क्र.७८/ उद्योग-७
या वाक्यामध्ये पुढीलप्रमाणे सुधारणा करण्यात येत आहे :
---------------------------------------------------------------------------------------------------
| “राज्यातील वार्षिक १०,००० मे. टन वापर असणा-या लघू, मध्यम व इतर उपक्रमांना कोळसा वितरण |
| करण्यासाठी शासन स्वतंत्र आदेशाद्वारे शासन नियुक्त यंत्रणेची विहीत कालावधीसाठी नियुक्ती करेल.” |
|  |
| २. संदर्भ क्र.०३ येथील दिनांक १२/०७/२०१६ च्या शासन निर्णयातील परि. क्र. ४.१ मध्ये “ उद्योग |
| घटकाने कोळसा वाटपाबाबतचे स्वयंघोषित प्रमाणपत्र द्यावे. उद्योग घटकांची तांत्रिक तपासणी ६ |
| महिन्यांच्या आत करणे बंधनकारक आहे' असे नमूद |  |
|  | या वाक्यामध्ये पुढीलप्रमाणे सुधारणा करण्यात येत आहे : |  |
| “ज्या उद्योग घटकाद्वारे मात्रा व क्षमता निर्धारण प्रमाणपत्र सादर करण्यात येईल त्याच उद्योग घटकाला |
| कोळसा वितरण करण्यात यावे.” |  |
|  |
| ३. संदर्भ क्र.०३ येथील दिनांक १२/०७/२०१६ च्या शासन निर्णयातील परि. क्र. ६.२ मध्ये ६.२ (ड) |
| नव्याने समाविष्ट करण्यात येत असून ते पुढीलप्रमाणे आहे: |  |
|  | “कोणत्याही उद्योग घटकाबाबत तक्रार प्राप्त झाल्यास शासन नियुक्त यंत्रणा त्या उद्योग |
| घटकाची तपासणी करेल व तपासणी नंतर कोळसाची अफरातफर आढळल्यास व तक्रार योग्य |
| उद्योग घटकाची सुरक्षा ठेव जप्त केली जाईल आणि जेवढ्या कोळसा मात्राची |
| अफरातफर सिध्द झाली आहे तेवढ्या कोळसा मात्राची किंमत त्या उद्योग घटकाकडून वसुल करण्यात |
| येईल. कोळश्याची रक्कम जमा न केल्यास पोलिसात तक्रार करण्यात येईल आणि त्या उद्योग घटकास |
| काळ्या यादीत टाकण्यात येईल.” |  |
| ४. उपरोक्त अनु. क्र. १ ते ३ येथील सुधारणा वगळता कोळसा वाटपाबाबत यापूर्वी दिनांक |
| १२/०७/२०१६ व दिनांक ०१/०१/२०१९ रोजीच्या शासन निर्णयातील उर्वरीत तरतूदी कायम राहतील, |
| याची सर्व संबंधित यंत्रणांनी नोंद घ्यावी. तसेच राज्यातील लघु, मध्यम व इतर उपक्रमांना कोळसा |
| वितरण अधिक पारदर्शक पद्धतीने करण्याबाबत शासन नियुक्त यंत्रणेकडून खालील तरतूदी प्राथम्याने |
| अंमलात आणाव्यात : |  |
| ।) शासन नियुक्त यंत्रणेने कोळसा वितरणासंदर्भात लवकरात लवकर सर्वसमावेशक ई-पोर्टलची |
|  | निर्मिती त्यांच्या तरतूदीतून करावी. त्यामध्ये राज्यातील उद्योग घटकांसाठी CIL यांच्याकडून |
|  | कोळशाच्या वाटपाकरीता आवंटन प्राप्त झालेला तपशील, उद्योगघटकाने करावयाची |
|  | ऑनलाईन अर्ज पध्दती, मात्रा व क्षमता निर्धारण प्रमाणपत्र अपलोड करणे, जिल्हा स्तरीय |
|  | वाटप समितीचा निर्णय, उद्योग घटकनिहाय कोळशाची मागणी, कोळसा वापराबाबतचा |
|  | तपशील इत्यादी बाबींचा समावेश असेल. तसेच उद्योग घटकाने किती कोळसा कोणत्या |
|  | खाणीतून, कोणत्या GPS प्रणालीयुक्त ट्रक ने प्राप्त केला, त्याकरीता किती रक्कम भरणा |
|  | केली त्याविषयक तपशील याची दर महिन्याला ई-पोर्टलवर माहिती अद्ययावत करण्याची |
|  | जबाबदारी उद्योग घटकाची राहील. त्यासाठी Code शासन नियुक्त यंत्रणा देईल. सदर ई |
|  | पोर्टल अद्ययावत ठेवण्याची जबाबदारी शासन नियुक्त यंत्रणेची राहील. |  |
|  |
| II) शासन नियुक्त यंत्रणेमार्फत वाटप करण्यात येणा-या कोळशाची वाहतूक करणा-या |
|  | वाहनांकरीता GPS प्रणाली लवकरात लवकर विकसित करण्यात येवून या प्रणाली आधारे |
|  | कोळसा खाणीपासून उपक्रमापर्यंत करण्यात येणारी कोळशाची वाहतूक सनियंत्रित |
|  | करण्याबाबत सविस्तर आराखडा शासन नियुक्त यंत्रणेने विकसित करावा. तसेच त्या |
|  | आराखड्याचा अंमल लवकरात लवकर करावा. या प्रयोजनार्थ होणारा खर्च शासन नियुक्त |
|  | यंत्रणेने त्यांच्या तरतूदीतून करावा. |  |
