शासन निर्णय सर्व शिक्षा अभियान या केंद्र पुरस्कृत योजनेंतर्गत सन २०१८-१९ या वित्तीय वर्षाकरीता मागणी क्र. टी-५, २२०२, सर्वसाधारण शिक्षण, पंचवार्षिक योजनांतर्गत योजना, ०१, प्राथमिक शिक्षण, ७९६ जनजाति क्षेत्र उपयोजना (०१) जनजाति क्षेत्र उपयोजनांतर्गत योजना राज्य योजनांतर्गत योजना, (०१) (०६) सर्व शिक्षा अभियान योजना ( ६० टक्के केंद्र हिस्सा ) लेखाशिर्ष (२२०२ आय४४१ ), ३१ सहायक अनुदाने (वेतनेत्तर) करिता एकूण रु. ११२.५० कोटी तरतूद अर्थसंकल्पित आहे. तर त्यास समवर्ती राज्य हिस्सा म्हणून (०१) (०७) सर्व शिक्षा अभियान योजना ( ४० टक्के राज्य हिस्सा ) लेखाशिर्ष ( २२०२ आय ४५२ ) ३१- सहायक अनुदाने (वेतनेत्तर) करिता एकुण रु. ७५.०० कोटी तरतूद अर्थसंकल्पित आहे.
२. वाचा क्र. २ येथील नस्तीवर प्रस्तावित केल्यानुसार मागणी क्र. टी-५, २२०२, सर्वसाधारण शिक्षण, पंचवार्षिक योजनांतर्गत योजना, ०१, प्राथमिक शिक्षण, ७९६ जनजाति क्षेत्र उपयोजनाए (०१) जनजाति क्षेत्र उपयोजनांतर्गत योजना राज्य योजनांतर्गत योजना, (०१) (०६) सर्व शिक्षा अभियान योजना (६० टक्के केंद्र हिस्सा ) (२२०२ आय४४१ ), ३१ सहायक अनुदाने (वेतनेत्तर) या लेखाशिर्षाखाली केंद्र शासनाने रु.३६.७२ कोटी ( रुपये छत्तीस कोटी बाहत्तर लाख ) फक्त इतका निधी वितरीत केला असुन त्यास समरुप राज्य हिस्सा म्हणून (०१) (०७) सर्व शिक्षा अभियान योजना ( ४० टक्के राज्य हिस्सा ) ( २२०२ आय४५२ ) ३१- सहायक अनुदाने (वेतनेत्तर) या लेखाशिर्षाखाली रु. २४.४८ कोटी ( रुपये चोवीस कोटी अठ्ठेचाळीस लाख फक्त ) इतका निधी अर्थसंकल्पीय वितरण प्रणालीवर वितरीत करण्यास याद्वारे शासनाची मान्यता देण्यात येत आहे.
प्रकरणी नियंत्रण अधिकारी / विभागप्रमुख यांनी खर्चासंबंधीच्या सध्याच्या शासकीय धोरणास अनुसरुन मंजूर आराखडयानुसार याच वर्षात सदर निधी खर्च होईल याची दक्षता घ्यावी. तसेच उपलब्ध करुन दिलेल्या तरतूदीतून झालेल्या खर्चाचा अहवाल, निधी उपयोगिता प्रमाणपत्र, लेखाशिर्षनिहाय / उपलेखाशिर्षनिहाय साध्य झालेले भौतिक उद्दिष्ट्ये इ. माहिती, संबंधित क्षेत्रिय अधिकारी यांनी आयुक्त, आदिवासी विकास, नाशिक; आदिवासी विकास विभाग व पदुम विभाग यांना प्रत्येक महिन्याच्या १० तारखेपर्यंत पाठवणेबाबत सूचना देण्यात यावी.

शासन निर्णय क्रमांकः बीयुडी-२०१८/प्र.क्र.०४/कार्यासन.६
३. नियंत्रक अधिकारी/ विभाग प्रमुख यांनी आदिवासी उपयोजना क्षेत्रात, अतिरिक्त आदिवासी उपयोजना क्षेत्रात व आदिवासी घटक कार्यक्रम क्षेत्राबाहेरील योजनांवर केलेला खर्च स्वतंत्रपणे नोंदविला जाईल याची दक्षता घ्यावी, जेणेकरुन शासनाला आदिवासी क्षेत्रातील व उपयोजने क्षेत्राबाहेरील योजनांवर होणा-या खर्चावर संनियंत्रण ठेवता येईल.
४. आदिवासी घटक कार्यक्रमांतर्गत उपलब्ध करुन दिलेली तरतूद फक्त आदिवासी लाभार्थ्यांकरिता व आदिवासी उपयोजना क्षेत्रांतर्गत खर्च करण्यात यावी. तसेच आदिवासी उपयोजना क्षेत्राबाहेर केलेली तरतूद त्या क्षेत्रातील आदिवासी वैयक्तीक लाभार्थ्यांकरिताच खर्च करण्यात यावी.
५. सदर शासन निर्णय वित्त विभागाच्या शासन परिपत्रक क्र. अर्थसं - २०१८ / प्र.क्र.६९ / अर्थ-३, दि.०२.०४.२०१८ नुसार प्राप्त अधिकार तसेच वित्त विभागाचा अनौपचारीक संदर्भ क्र. २९९/अर्थोपाय, दि.१८.०५.२०१८ अन्वये प्राप्त सहमतीनुसार निर्गमित करण्यात येत आहे.
