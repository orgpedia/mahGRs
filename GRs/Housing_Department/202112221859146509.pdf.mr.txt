दिनांक:२२ डिसेंबर, २०२१.
वाचा:- १) शासन निर्णय गृहनिर्माण विभाग क्रमांक : संकीर्ण -२०२०/प्र.क्र.१८/झोपनि-२ दि. १४/०७/२०२१ २) शासन अधिसुचना गृहनिर्माण विभाग क्रमांक संकीर्ण-२०२०/प्र.क्र.१८/झोपनि-२ दि. १२/०८/२०२१

| मुंबई शहर व उपनगरातील सर्व धारणाधिकार व मालकी हक्काच्या जमिनी, केंद्र शासनाच्या संबंधित |
| विभागाने त्यांच्या मालकीच्या झोपडपट्टीव्याप्त जागेचे सर्वेक्षण करण्यास ना-हरकत दिलेल्या जमिनी, मुंबई |
| उपनगर जिल्हातील भारतीय विमान पतन प्राधिकरणाच्या जमिनीवरील झोपडपट्टीचे क्षेत्र आणि मिठी नदी |
| विस्तारीकरण टप्पा क्रमांक २ ने बाधित झोपडपट्टीचे क्षेत्रातील झोपु योजनेतील झोपडीधारकांची पात्रता/अपात्रता |
| तपासणे यासाठी पुर्वी १८ उपजिल्हाधिकारी (अतिक्रमण/निष्कासन) हे सक्षम प्राधिकारी म्हणून कार्यरत होते. |
| तसेच, झोपु योजनेतील झोपडीधारकांची पात्रता / अपात्रता तपासण्यासाठी संबंधित उपजिल्हाधिकारी |
| (अतिक्रमण/निष्कासन) यांचेकडे प्रस्ताव पाठविणे, सदरचे प्रस्ताव त्यांचेकडून परत घेणे, शासनाकडे याबाबत |
| माहिती पाठविणे अशाप्रकारचे समन्वयाचे कामकाज झोपुप्रा, मुंबई यांचेकडील कार्यरत ३ उपजिल्हाधिकारी |
| यांचेमार्फत करण्यात येत होते. अशाप्रकारे एकाच कामासाठी वेगवेगळ्या प्रशासकीय यंत्रणा काम करत होत्या. |
| सबब, प्रशासकीय कामामध्ये एकवाक्यता व सूसुत्रता आणण्यासाठी शासन गृहनिर्माण विभागाच्या |
| दि.१४/०७/२०२१ रोजीच्या शासन निर्णयान्वये प्रशासकीय यंत्रणा नव्याने पुनर्रचित करून झोपडपट्टी पुनर्वसन |
| प्राधिकरण, मुंबई यांचेमार्फत वेतन अदा करण्यात येणारे ७ उपजिल्हाधिकारी (अतिक्रमण/निष्कासन) व झोपडपट्टी |
| पुनर्वसन प्राधिकरण, मुंबई यांचेकडील ३ उपजिल्हाधिकारी यांना महाराष्ट्र झोपडपट्टी क्षेत्र (सुधारणा, निर्मूलन व |
| पुनर्विकास) अधिनियम, १९७१ मधील कलम ३ च्या पोट-कलम (१), कलम ३Y च्या उप कलम (१) अन्वये सक्षम |
| प्राधिकारी घोषीत करण्यात आले आहे. त्याचबरोबर सदर सक्षम प्राधिकारी व अपिलीय प्राधिकारी तथा अपर |
| जिल्हाधिकारी हे मुख्य कार्यकारी अधिकारी, झोपुप्रा यांच्या प्रत्यक्ष नियंत्रणाखाली आणण्यात आले. तसेच, शासन |
| अधिसुचना गृहनिर्माण विभाग क्रमांक संकीर्ण-२०२०/प्र.क्र.१८/झोपनि-२, दि.१२/०८/२०२१ अन्वये प्रत्येक सक्षम |
| प्राधिकारी यांचे कार्यक्षेत्र आणि अपिलीय प्राधिकारी निश्चित करण्यात आले आहेत. |
| २. महाराष्ट्र झोपडपट्टी क्षेत्र (सुधारणा, निर्मूलन व पुनर्विकास) अधिनियम, १९७१ मधील कलम ३ च्या पोट |
| कलम (१), कलम ३Y च्या उप कलम (१) अन्वये सक्षम प्राधिकारी घोषीत करण्यात आले असल्यामुळे पुर्वीच्या १८ |
| उपजिल्हाधिकारी (अतिक्रमण/निष्कासन) कार्यालयात मंजूर असलेल्या तहसिलदार, नायब तहसिलदार, अव्वल |
| कारकून, भुमापक, वसुली कारकून/महसुली कारकून / महसुली सहाय्यक या संवर्गातील पदांपैकी काही पदे |
| नव्याने घोषीत करण्यात आलेल्या सक्षम प्राधिकारी, झोपुप्रा, मुंबई या कार्यालयांकडे वर्ग करण्याची बाब |
| शासनाच्या विचाराधीन होती. |

शासन परिपत्रक क्रमांकः संकीर्ण-२०२१/प्र.क्र.१२०/झोपनि-२

| शासन परिपत्रक : |
| महाराष्ट्र झोपडपट्टी क्षेत्र (सुधारणा, निर्मूलन व पुनर्विकास) अधिनियम, १९७१ चे कलम ३ (K) नुसार |
| शासनास प्राप्त अधिकारानुसार पुर्वीच्या १८ उपजिल्हाधिकारी (अतिक्रमण/निष्कासन) कार्यालयात मंजूर |
| असलेल्या तहसिलदार, नायब तहसिलदार, अव्वल कारकून, भुमापक, वसुली कारकून / महसुली कारकून / |
| महसुली सहाय्यक या संवर्गातील पदांपैकी सोबत जोडलेल्या परिशिष्ट - १ ते ५ मध्ये नमूद केल्याप्रमाणे पदे |
| नव्याने घोषीत करण्यात आलेल्या सक्षम प्राधिकारी, झोपुप्रा, मुंबई या कार्यालयांकडे खालील अटी व शर्तींच्या |
| अधीन राहून वर्ग करण्यात येत आहेत : |
---------------------------------------------------------------------------------------------------------
-----------------------------------------------------------------------------------------------------
| अ) सक्षम प्राधिकारी, झोपुप्रा, मुंबई या कार्यालयांकडे सोबत जोडलेल्या परिशिष्ट - १ ते ५ मध्ये नमूद |
| केल्याप्रमाणे वर्ग करण्यात येत असलेल्या तहसिलदार, नायब तहसिलदार, अव्वल कारकून, |
| भुमापक, वसुली कारकून/महसुली कारकून/महसुली सहाय्यक या संवर्गातील पदांचे वेतन हे |
| शासनाकडून पुर्वीप्रमाणेच संबंधीत लेखाशिर्षातून अदा करण्यात येईल. |
| आ) सक्षम प्राधिकारी, झोपुप्रा, मुंबई या कार्यालयांकडे वर्ग करण्यात येत असलेल्या तहसिलदार, नायब |
| तहसिलदार, अव्वल कारकून, भुमापक, वसुली कारकून/महसुली कारकून/महसुली सहाय्यक या |
| संवर्गातील पदांचे वेतन अदा करण्यासाठी सध्या जे आहरण व संवितरण प्राधिकारी व नियंत्रक |
| प्राधिकारी आहेत, तेच प्राधिकारी पुढील आदेश होईपर्यंत आहरण व संवितरण प्राधिकारी व नियंत्रक |
| प्राधिकारी म्हणून राहतील. |
| इ) संबंधित सक्षम प्राधिकारी, झोपुप्रा, मुंबई यांनी त्यांच्या कार्यालयात वर्ग करण्यात आलेल्या पदांवर |
| कार्यरत अधिकारी/कर्मचारी यांची सेवाविषयक माहिती स्वतंत्रपणे ठेवावी. |
| ई) सक्षम प्राधिकारी, झोपुप्रा, मुंबई या कार्यालयांकडे वर्ग करण्यात येत असलेल्या तहसिलदार, नायब |
| तहसिलदार, अव्वल कारकून, भुमापक, वसुली कारकून/महसुली कारकून/महसुली सहाय्यक यांचे |
| गोपनीय अहवाल हे संबंधित सक्षम प्राधिकारी लिहितील. |
