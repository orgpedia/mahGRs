| अनुक्रमे १००० मांसल कुक्कुट पक्षी संगोपनाव्दारे कुक्कुट पालन व्यवसाय सुरु करण्यासाठी पायाभूत |
| सुविधा पुरविणे आणि १० शेळया / मेंढया + १ बोकड / नर मेंढा वाटपाच्या योजनेस प्रशासकीय |
| मान्यता प्रदान करण्यात आलेली असून, या दोनही योजना राज्यात राबविण्यात येत आहेत. |
| २. उपरोक्त दोनही योजनांमध्ये निवड झालेल्या लाभार्थ्यांस प्रत्यक्ष पायाभूत सुविधा |
| उभारण्यासाठी तसेच शेळी / मेंढी गटाची प्रत्यक्ष खरेदी करण्यासाठी कालावधी विहीत करण्यात |
| आलेला नसल्यामुळे सदर कालावधी विहीत करण्याची बाब शासनाच्या विचाराधीन होती. |

शासन निर्णय : ३. वर नमुद वस्तुस्थिती विचारात घेता खालीलप्रमाणे शासन निर्णय निर्गमित करण्यात येत आहे.
(i) राज्यस्तरीय नाविन्यपूर्ण योजनेंतर्गत १००० मांसल कुक्कुट पक्षी संगोपनाव्दारे कुक्कुट पालन
----------------------------------------------------------------------------------------------
| १००० मांसल कुक्कुट पक्षी संगोपनाव्दारे कुक्कुट पालन |
| व्यवसाय सुरु करण्यासाठी पायाभूत सुविधा पुरविणे या योजनेंतर्गत संबंधित जिल्हा वर्धन |
| उपायुक्त यांच्याकडून लाभार्थ्यांची निवड झाल्याबाबत पत्राव्दारे कळविल्याच्या दिनांकापासून तीन |
| महिने कालावधीत १०० टक्के पायाभूत सुविधा उभारणे बंधनकारक राहील, असे न केल्यास जिल्हा |
| पशुसंवर्धन उपायुक्त यांनी संबंधितास ३० दिवसांची अंतिम मुदत द्यावी आणि याही कालावधीत |
| संबंधितांनी पायाभूत सुविधांची उभारणी न केल्यास संबंधितांना या योजनेत लाभ घ्यावयाचा नाही, |
| असे गृहित धरुन प्रतिक्षा यादीमधील पुढील लाभार्थ्यांस लाभ देण्याची कार्यवाही करावी. |
| (ii) सदर योजनेंतर्गत सन २०२१-२२ पुर्वी लाभ मंजुर करण्यात आलेल्या ज्या लार्भार्थ्यांनी काही |
| अपरिहार्य कारणास्तव अदयाप पायाभुत सुविधांची उभारणी केलेली नाही व अशा लाभार्थ्यांना |
| अखर्चित निधी शिल्लक असल्यास अशा प्रकरणी शेवटची संधी म्हणून ३० दिवसांची अंतिम मुदत द्यावी |
| आणि याही कालावधीत संबंधितांनी पायाभूत सुविधांची उभारणी न केल्यास संबंधितांना या योजनेत |
