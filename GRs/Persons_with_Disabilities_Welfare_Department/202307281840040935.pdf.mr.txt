५) शासन निर्णय, दिव्यांग कल्याण विभाग, क्रमांक दिव्यांग-२०२३/प्र.क्र.५६/दि.क.२, दि.२३ मे, २०२३.
शासन परिपत्रक: विविध प्रवर्गातील दिव्यांग बांधवांच्या अडचणी सोडविण्यासाठी जिल्हा स्तरावर शासनाचे सर्व विभागाचे अधिकारी, सर्व शासकीय यंत्रणा एकाच ठिकाणी उपस्थित राहून दिव्यांगांच्या तक्रारी जाणून घेऊन त्याच ठिकाणी कामाचा निपटारा करता येईल. याकरीता या विभागाचा शासन निर्णय दिनांक २३ मे २०२३ नुसार "दिव्यांग कल्याण विभाग दिव्यांगाच्या दारी" हे अभियान राबविण्यात येत आहे. सोबत जोडलेल्या “प्रपत्र ब” प्रमाणे तारखेनिहाय राज्यातील सर्व जिल्हयांत अभियानाचे आयोजन करण्यात येणार आहे. त्यानुसार सदर अभियानाची जिल्हा स्तरावर प्रभावीपणे अंमलबजावणी होणेसाठी सर्वसाधारणपणे प्रमाणित कार्यपद्धती SOP (Standard operating procedure) असल्यास जास्तीत जास्त उपयुक्तता राहणार आहे. यासाठी "दिव्यांग कल्याण विभाग दिव्यांगाच्या दारी" हे अभियान जिल्हा स्तरावर राबविताना खालीलप्रमाणे सर्वसाधारण कार्यपद्धतीचा अवलंब करण्यात यावा.

३. सद्यस्थितीत पावसाचे दिवस असल्याने कार्यक्रमासाठी मोठ्या हॉलची व्यवस्था करावी.

| ब) विभाग प्रमुखांच्या कर्तव्य व जबाबदाऱ्या : |
| १. प्रत्येक विभाग प्रमुखांनी त्यांच्या विभागाच्या योजनांचा लाभ दिव्यांग लाभार्थ्यांना अभियानाच्या |
| कार्यक्रमाच्या दिवशी कसा देता येईल याची रुपरेषा निश्चित करुन जिल्हाधिकारी यांच्या अध्यक्षतेखाली |
| होणाऱ्या बैठकीमध्ये मांडावी व अंतिम करुन घ्यावी. |
| २. प्रत्येक विभागाने आपल्या योजनांसाठी आवश्यक असलेल्या कागदपत्रांची माहिती दिव्यांग लाभार्थ्यांना |
| मिळणेसाठी स्थानिक वृत्तपत्रांमध्ये बातमी द्यावी व आवश्यक जनजागृती करावी. जेणेकरुन |
| अभियानाच्या कार्यक्रमा दिवशी दिव्यांग लाभार्थी योजनेसाठी आवश्यक असलेल्या कागदपत्रांसहीत |
| उपस्थित राहतील व त्यांना योजनेचा लाभ घेणेसाठी अर्ज सादर करता येईल. शक्य झाल्यास |
| जिल्हाधिकारी व मुख्य कार्यकारी अधिकारी (जिल्हा परिषद) यांनी आकाशवाणीवर त्यांच्या |
| मुलाखतींतून या बाबींवर प्रक्षेपण घडवून आणावे. |
| ३. प्रत्येक विभागाने आपल्या योजनांची माहिती पुस्तिका/घडीपत्रिका/ पॉम्पलेट जास्तीचे छापून घ्यावेत |
| व कार्यक्रमादिवशी दिव्यांग लाभार्थ्यांना वाटप करण्यात यावेत. |
| ४. दिव्यांगाच्या विविध योजना तसेच शासन निर्णयांचे ई-बुक चे क्युआर कोड तयार करुन कार्यक्रम स्थळी |
| याबाबत माहिती व प्रसिध्दी करावी. |

| क) कार्यक्रमाचे स्वरुप : |
| १. अभियानाच्या कार्यक्रमादिवशी जिल्ह्याच्या ठिकाणी दिव्यांग लाभार्थी पोहचल्यानंतर बस |
| स्थानकापासून कार्यक्रमाच्या ठिकाणी पोहचण्यासाठी वाहनांची सुविधा उपलब्ध करुन द्यावी. |
| २. दिव्यांग लाभार्थी कार्यक्रमाच्या ठिकाणी पोहचल्यानंतर प्रत्येक दिव्यांग लाभार्थ्याची नोंदणी करुन |
| घेण्यात यावी.यासाठी दिव्यांगत्वाच्या प्रवर्गनिहाय नोंदणी करुन घेणेसाठी सुविधा व मनुष्यबळ उपलब्ध |
| करुन द्यावे. एकाच ठिकाणी गर्दी होऊ नये म्हणून नोंदणी काऊंटर्स पुरेशा संख्येने उपलब्ध करुन |
| द्यावीत. नोंदणीसाठी ऑनलाईन पद्धतीने संगणकीय नोंदणीस प्राधान्य द्यावे. |
| ३. वेगवेगळया विभागांच्या विविध योजनांचा लाभ घेणेसाठी दिव्यांग लाभार्थ्यांना वेगवेगळे अर्ज करणेसाठी |
| प्रत्येक विभागाने त्यांचे विहीत नमुन्यातील अर्ज कार्यक्रमादिवशी लाभार्थ्यांना उपलब्ध करुन द्यावेत. |
| ४. शासनांच्या योजनांचे अर्ज मॅन्युअली तसेच काही योजनांचे अर्ज हे ऑनलाईन पद्धतीने भरावे लागतात. |
| याकरिता कार्यक्रमाच्या दिवशी प्रत्येक विभागाच्या स्टॉलवर दिव्यांग व्यक्ती लाभ घेऊ इच्छिणाऱ्या |
| योजनेचा मॅन्युअली/ऑनलाईन अर्ज भरण्यास तो दिव्यांग व्यक्ती सक्षम नसल्यास अर्ज भरण्यास मदत |
| करण्याकरिता स्वयंसेवक तसेच ऑनलाईन अर्ज भरण्याकरिता संगणक व संगणक ऑपरेटर उपलब्ध |
