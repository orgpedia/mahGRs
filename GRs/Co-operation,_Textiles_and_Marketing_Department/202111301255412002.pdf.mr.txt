| प्रस्तावना : |
| राज्यातील सहकारी उपसा जलसिंचन योजनांच्या प्रकल्प खर्चामध्ये गेल्या काही वर्षात |
| सातत्याने वाढ होत असून, त्यामुळे अल्प व सिमांत शेतकरी सभासदच नव्हे तर मध्यम व मोठया |
| शेतकऱ्यांना सुध्दा वाढता प्रकल्प खर्च करणे आर्थिकदृष्या अडचणीचे झाले आहे. शासनाच्या खर्चाने |
| ज्या लहान मोठया पाटबंधारे योजना हाती घेण्यात येतात त्या योजनांच्या लाभक्षेत्रातील |
| शेतकऱ्यांना कोणत्याही खर्चाविना पाण्याची सोय उपलब्ध होते. या उलट इतर भागातील |
| शेतकऱ्यांनी स्वत:च उपसा जलसिंचनाची व्यवस्था करण्याचे ठरविल्यास त्यांच्या योजनांचा संपूर्ण |
| खर्च त्यांना स्वत:च करावा लागतो. त्यामुळे सहकारी उपसा जलसिंचन योजनांच्या लाभधारक |
| सभासदांवरील खर्चाचा बोजा कमी करुन तो काही प्रमाणात शासनाकडून उचलला जावा, या |
| दृष्टीकोनातून उपरोक्त शासन निर्णय, दिनांक १९.११.१९९४ अन्वये सदर शासन निर्णयासोबत |
| जोडलेल्या परिशिष्ट “अ” मधील अटींच्या अधिन राहून वित्त विभागाने उपलब्ध करुन दिलेल्या |
| नियतव्ययामधून सन २०२१-२२ या आर्थिक वर्षात सहकारी उपसा जलसिंचन योजनांना |
| अर्थसहाय्य देण्याची बाब शासनाच्या विचाराधीन होती. |
| शासन निर्णय - |
| सहकार आयुक्त व निबंधक, सहकारी संस्था, महाराष्ट्र राज्य, पुणे यांच्याकडून सदर |
| शासन निर्णयानुसार अनुदान मंजूरीसाठी प्राप्त झालेल्या प्रस्तावांपैकी सन २०२१-२२ या वित्तीय |
| वर्षासाठी ०८ सहकारी उपसा जलसिंचन संस्थांना रु. १,४५,५५,०००/- (रुपये एक कोटी |
| पंचेचाळीस लक्ष पंचावन्न हजार फक्त) इतके अनुदान मंजूर करुन वितरीत करण्यास शासन मान्यता |
| देण्यात येत आहे. |
| २. सोबतच्या परिशिष्ट ‘अ’ मधील ०८ सहकारी उपसा जलसिंचन संस्थांना देण्यात आलेली |
| मंजूरी, प्रस्तावासोबत सहकार आयुक्त, पुणे यांच्याकडून सादर करण्यात आलेल्या माहितीच्या |
| आधारे देण्यात आलेली असल्याने सादर केलेल्या कागदपत्रात तसेच माहितीत काही विसंगती |
| आढळून आल्यास सदरहु रक्कम संबंधीत संस्थांकडून व्याजासह एकरकमी वसूल केली जाईल या |
| अटींच्या अधिन राहील. |

| ३. वरीलप्रमाणे मंजूर करण्यात आलेल्या अर्थसहाय्याचा रकमेचा धनादेश, ज्या बँक/वित्तीय |
| संस्थांकडून सहकारी उपसा जलसिंचन संस्थेने प्रकल्प खर्चाकरीता कर्ज घेतलेले असेल, त्या |
| बँकेच्या/वित्तीय संस्थेच्या व्यवस्थापक/प्रबंधक यांचे नावे काढण्यात यावा. तसेच सदर धनादेश |
| प्राप्त झाल्यानंतर बँकेने / वित्तीय संस्थेने त्यांना धनादेशाद्वारे प्राप्त झालेली अर्थसहाय्याची रक्कम |
| संबंधीत सहकारी उपसा जलसिंचन संस्थेकडून त्यांना येणे असलेल्या कर्जाच्या रकमेपोटी |
| समायोजित करावी. |
| ४. सन २०२१-२२ या आर्थिक वर्षात या योजनेकरीता उपलब्ध तरतुदीमधून |
| रु.१,४५,५५,०००/- (रुपये एक कोटी पंचेचाळीस लक्ष पंचावन्न हजार फक्त) इतकी रक्कम |
| परिशिष्ट ‘अ’ प्रमाणे सहकार आयुक्त व निबंधक, सहकारी संस्था, महाराष्ट्र राज्य, पुणे यांचे |
| कार्यालय स्तरावर बी. डी. एस. वर उपलब्ध करुन देण्यात आलेली आहे. या योजनेवरील रक्कम |
| खर्ची घालण्याकरीता सहकार आयुक्त व निबंधक, सहकारी संस्था, महाराष्ट्र राज्य, पुणे यांना |
| नियंत्रण अधिकारी म्हणून प्राधिकृत करण्यात येत आहे. |
| ५. निधी वितरीत केल्यानंतर किंवा त्यापूर्वी प्रत्यक्ष उपसा योजनेची पहाणी करुन निधीचा योग्य |
| वापर झाल्याची खात्री करावी. तसेच लाभार्थी शेतकऱ्यांची पण प्रत्यक्ष भेट घ्यावी. त्यामध्ये काही |
| चुकीचे होऊ नये याची दक्षता घ्यावी. |
| ६. वरीलप्रमाणे कार्यवाही न झाल्यास संस्थेबरोबर तपासणी अधिकाऱ्यावर देखील जबाबदारी |
| निश्चित करण्यात यावी. |
| ७. शासनाच्या वतीने सदर रक्कम कोषागारातून काढून संबंधीत बँक / वित्तीय संस्थेस |
| धनादेशाद्वारे अदा करण्यास लेखाधिकारी, सहकार आयुक्त व निबंधक, सहकारी संस्था, महाराष्ट्र |
| राज्य, पुणे यांना प्राधिकृत करण्यात येत असून त्यांना आहरणाचे व संवितरणाचे अधिकार प्रदान |
| करण्यात येत आहेत. त्यांनी सदरचा खर्च सन २०२१-२२ या आर्थिक वर्षातील |
| रु. १,४५,५५,०००/- (रुपये एक कोटी पंचेचाळीस लक्ष पंचावन्न हजार फक्त) तरतुदीतून मासिक |
| निधी विवरणपत्रानुसार भागवावा. लेखाधिकारी, सहकार आयुक्त व निबंधक, सहकारी संस्था, |
| महाराष्ट्र राज्य, पुणे यांनी वरीलप्रमाणे काढलेली रक्कम संबंधीत बँकेस / वित्तीय संस्थेस त्यांच्या |
| कर्जाच्या रकमेपोटी अदा केल्यानंतर त्याबाबतचा पूर्तता अहवाल १५ दिवसांत शासनास सादर |
| करावा. |
