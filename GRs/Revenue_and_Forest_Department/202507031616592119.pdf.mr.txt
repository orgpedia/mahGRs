| प्रस्तावना:- |

| आपत्ती व्यवस्थापन प्रभागांतर्गत, जिल्हा आपत्ती व्यवस्थापन प्राधिकरणाची आपत्ती विषयक कामकाज व |
| विषयक धोरण अंमलबजावणीमध्ये जिल्हाधिकारी तथा अध्यक्ष, जिल्हा आपत्ती व्यवस्थापन प्राधिकरण यांना |
| आपत्ती अनुषंगे कामकाज हाताळण्यासाठी जिल्हा आपत्ती व्यवस्थापन अधिकारी (DDMO) संवर्गात ३६ पदे तसेच |
| विभागीय स्तरावर आपत्ती विषयक कामकाजामध्ये समन्वय साधण्यासाठी विभागीय आपत्ती व्यवस्थापन |
| समन्वयक संवर्गात (DDMC) ६ पदे अशी एकूण ४२ पदे संदर्भीय शासन निर्णयान्वये कायमस्वरुपी कंत्राटी |
| तत्वावर निर्माण करण्यात आली आहेत. |
| सदर पदांना सन २०२१ पासून रू.४५,०००/- प्रतिमाह इतके मानधन अनुज्ञेय आहे. आपत्ती व्यवस्थापन |
| प्राधिकरणाच्या विविध आपत्ती अनुषांगिक कामकाजामध्ये मोठ्या प्रमाणावर वाढ तसेच कार्यक्षेत्राचा विस्तार |
| झालेला आहे. तसेच राष्ट्रीय आपत्ती व्यवस्थापन प्राधिकरणाच्या नियमित स्वरुपात आपत्ती अनुषांगिक नवनवीन |
| धोरण व योजनांची अंमलबजावणी राज्य आपत्ती व्यवस्थापन प्रभागाकडून करावी लागते. सदर योजना व धोरणांची |
| अंमलबजावणीमध्ये सदर जिल्हा आपत्ती व्यवस्थापन अधिकारी ब विभागीय आपत्ती समन्वयक अधिकारी हे |
| महत्वाची भूमिका बजावतात. |
| सदर पदांची स्थानिक पातळीवर आपत्ती व्यवस्थापनामधील महत्वाची भूमिका तसेच वाढती महागाई दर |
| विचारात घेऊन संदर्भ क्र.२ अन्वे राज्य कार्यकारी समितीच्या दि.९.०९.२०२४ रोजी झालेल्या बैठकोत या पदांच्या |
| मानधनामध्ये रू.४५०००/- वरून रू. ७५०००/- प्रतिमाह इतको वाढ करण्यास मान्यता दिली. समितीच्या या |
| निर्णयास राज्य आपत्ती व्यवस्थापन प्राधिकरणानाच्या संदर्भ क्र. अन्वे दि. ४.०६.२०२५ रोजी झालेल्या बेठकोत |
| सहमती दिली. त्यानुसार सदर पदांच्या मानधनामध्ये बाढ करण्याची बाब शासनाच्या विचाराधीन होती. |

२. सदर मानधन वाढ दिनांक ०१/०४/२०२५ पासून अनुज्ञेय राहील.
सदर प्रयोजनाबर होणार खर्च मागणी क्र. सो-६,२२४५,.नैसर्गिक आपत्तीच्या निवारणासाठी सहाय, ८० सर्वसाधारण (०१)(०१) महाराष्ट्र राज्य आपत्ती व्यवस्थापन प्राधिकारणाचे कार्यालय स्थापन करणे व फर्निचर साधन सामुग्री सह सूसज्य करणे (२२४५१७६१) १०-कंत्राटी सेवा या लेखशीर्षाखाली मंजूर अर्थसंकल्पीय तरतुदीतून भागविण्यात यावा.
सदर शासन निर्णय नियोजन विभागाच्या अनोप.सं.क्र. १००५/नियो दि. ०५/०३/२०२५ व वित्त विभाग अनोप.सं.क्र. २११/२०२५/व्यय-९ दि. २५/०३/२०२५ Bq प्राप्त सहमती व मान्यतेने निर्गमित करण्यात येत आहे.
